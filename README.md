# Personal Resume Website

This project is a personal resume website built with React, TypeScript, and Vite. It's designed to be a single-page application that showcases skills, experience, and projects in a clean, modern interface. The resume is fully responsive and features a print-friendly layout.

## ✨ Features

- **Built with Modern Tools**: [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), and [Vite](https://vitejs.dev/) for a fast and efficient development experience.
- **Styling**: Styled with [Tailwind CSS](https://tailwindcss.com/) and [shadcn/ui](https://ui.shadcn.com/) for a sleek and customizable design.
- **Internationalization (i18n)**: Supports both English and Chinese, with language data managed in JSON files.
- **Print-Friendly**: Includes specific styles to ensure the resume looks great when printed.
- **Component-Based**: Structured with reusable React components for easy maintenance and updates.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repo:
   ```shell
   git clone https://github.com/your-username/your-repository.git
   ```
2. Navigate to the project directory:
   ```shell
   cd your-repository
   ```
3. Install NPM packages:
   ```shell
   npm install
   ```

### Running the Development Server

To start the Vite development server, run:

```shell
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser. The page will reload automatically if you make edits.

## 📦 Deployment

This project is ready to be deployed to any static site hosting service.

### Building for Production

To create a production build, run:
```shell
npm run build
```
This command bundles the application into the `dist/` directory, which can then be deployed.

### Deploying with Vercel

You can easily deploy your project with [Vercel](https://vercel.com).

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/your-repository)

Alternatively, use the [Vercel CLI](https://vercel.com/download) from your terminal:
```shell
vercel
```
